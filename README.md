# Recipes-App-React-Native


This is basically an app where you can find the recipes of delicious dishes.

# Features

* Home page
* Cooking Time
* Ingredients
* Categories
* Recipes Details Screen
* Photo Gallery
* Detailed Description
* Ingredients Button
* Ingredients List
* Recipes with Ingredients
* Search
* Drawer Menu
* Navigation Bar
